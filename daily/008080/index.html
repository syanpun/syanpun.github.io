瞎扯

好想要自己整一个贼漂亮的网站呀，也只是想了好久没有动手做。

大学时也买过域名服务器，折腾过 WordPress 和其他一些动态 / 静态博客（一直在找主题写不出文章）。

这下好了，突然看到一个大佬整的超级好看的 Hexo 主题好生羡慕，nó 不信大家可以去看看 云游君的小站 ，看了一下大佬的博客和 GitHub 本菜鸡是十分佩服（给大佬献上膝盖🙈）。

昨天下单了个域名 syanpun.me （md语法都忘记了，写个链接两次 啊不三次了 啊不变成四次了 把圆括号和方括号写反，逃~），先看看能不能整几篇文章出来发到 GitHub 上，有几篇文章之后再去搞一台云服务器吧😭

试图回到原主题

008080 真是太好看了呀，得想个办法把它整成主色调

前端样式实在是太难了，我也不知道怎么回事找着配色突然进到了 Element UI 官网（为什么要加个链接我也不知道，可能是想让后面整好主题色之后颜色多一些。。）（又想起来之前也花过时间在这里换皮，哈哈还用在了我的毕业设计上），然后又花了几个小时在这找看着顺眼的颜色在这里换皮，希望能祝我后面一臂之力。

（为什么我要加这么多括号，花里胡哨一些 显得活泼一些 写文章肚子里没有货然后写着突然想吐槽和凑字数😭）

啊这配色 乍一眼 乍几眼看都好好看啊，不过上次毕设用 008080 作主色调看过一段时间后就觉得有点平淡不耐看了，但是今天看了好多眼还是好好看呀~
